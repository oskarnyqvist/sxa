# Jones in the Fast Lane — Konfigurerbar spelmotor

## Vad är det?

Webbaserad version inspirerad av "Jones in the Fast Lane" — ett livssimulerings/brädspelsliknande spel där man kör runt, jobbar, handlar, utbildar sig etc.

## Grundidé

- Inte bara ett spel utan en konfigurerbar motor
- Via konfiguration (JSON/YAML?) kan man skapa helt andra spel med samma grund
- Olika teman, platser, mål, resurser — allt styrt av config
