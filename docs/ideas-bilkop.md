# Bilköp — Sidekick för begagnatköp

## Vad är det?

Ett personligt projektverktyg för den som ska köpa begagnad bil. Inte en marknadsplats — ett verktyg för att hantera *processen*: samla kandidater, jämföra, föra anteckningar, och hålla koll på var man är med varje bil.

## Kärnfunktioner

- **Kanban-board** med stadier: Intressant → Kontaktad → Provkörning → Bud → Köpt/Avförd
- **Bilkort** per kandidat: länk till annons, pris, mil, år, modell, egna anteckningar, bilder
- **Checklistor** — mallar för provkörning, dokumentkoll, saker att fråga säljaren
- **Jämförelsevy** — ställ kandidater sida vid sida på nyckeldata
- **Tidslinje/logg** — vad har hänt med varje bil (ringde säljaren, bokade provkörning, etc)

## Vad det inte är

- Ingen marknadsplats eller sökmotor (Blocket/Bytbil finns redan)
- Ingen prisvärderings-tjänst
- Ingen inloggning krävs — datan lever lokalt eller i en enkel session
